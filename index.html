Stock Brokerage

This project is an academic project that was developed for my Web Programming Languages course at UTD.

• Developed a scalable web application with the base functionality of buying and selling stocks
• Exposed REST APIs with JSON to fetch stock prices, process buy and sell orders
• Implemented asynchronous AJAX calls to Stock brokerage web service to receive and process data before displaying
• Secured the application using TLS/SSL
• Performed request/response compression using gzip and implemented caching via Memcached for high performance

Exercise Tracker

Problem Statement: While I'm not binging away on (un)healthy snacks, I try to squeeze in some exercise time. However, I somehow always lose track of the duration and/or type of routine I followed! :(

To help myself and others like me who have trouble keeping track of their exercise routines, I developed a tracker application using MERN stack. This application allows multiple users to maintain log records for their exercise routines. I also incorporated update and delete functionality to allow users to modify their log records.

Shopping List

Problem statement: Shopping can become a real task when you've got a ton of items to purchase or are keenly looking out for newly launched products! :/

To help ease the logistics of shopping, I created a Single Page Application using Angular which allows visitors to browse through a catalogue of products, share details of the product and receive notifications for sale of new products
